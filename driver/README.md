# PhoneCam Virtual Camera Driver

> ⚠️ **Under Development** — Driver source code will be published here soon.

The compiled driver installer is available as a download on the
[Releases page](https://github.com/Sudo0xSajal/phonecam/releases/latest/download/PhoneCam-Driver.zip)
once it's ready.

## What it does

Installs a virtual camera device on Windows so apps like Zoom, Teams,
and Google Meet can see "PhoneCam Connect" in their camera dropdown.

## Requirements

- Windows 10 / 11 (64-bit)
- Administrator access (one-time install)
- No reboot required
