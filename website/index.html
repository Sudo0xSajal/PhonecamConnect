Four simple steps and your phone is a professional webcam. No technical knowledge needed.

02

Capabilities

Everything You Need

Not just a stream. A complete virtual camera solution.

🎥

Native Windows Webcam

"PhoneCam Connect" appears in every app's camera picker — Zoom, Teams, WhatsApp, Meet, Discord, OBS — without OBS as middleware.

📱

Multi-Device Support

Connect multiple phones at once. Switch the active webcam source per-device from the dashboard. Each phone has independent quality settings.

⚙️

Quality Control

Per-device: resolution (480p → 4K), FPS (15/24/30/60), and JPEG quality (40–100%). Changes sync instantly to phone and all UI elements.

🔄

Flip, Rotate, Mirror

0°/90°/180°/270° rotation with smooth preview. Aspect ratio: Auto-detect, 16:9, 9:16, 4:3, 1:1, 21:9. Mirror mode for selfie style.

📊

Live Analytics

Real-time FPS counter, latency, bitrate, dropped frames chart. See exactly how your stream is performing on the Analytics page.

🔁

Auto-Reconnect

Heartbeat every 10s detects dead connections. On disconnect, auto-switches to next available phone. FFmpeg crashes auto-restart up to 3×.

🔐

Secure Architecture

contextBridge IPC with channel whitelists. contextIsolation enabled. URL validation on external links. CSP headers enforced.

🔦

Camera Controls

Toggle phone torch, flip front/back camera, adjust mic, toggle noise suppression — all from the desktop app without touching your phone.

📡

Always-On MJPEG

Port 7781 serves a standard MJPEG stream. Add it to OBS as a Browser Source, open in VLC, or embed in any browser — zero setup.

Get Started

Up and Running in 2 Minutes

1

Download PhoneCam Connect

Download the Windows desktop app. Run the installer — no configuration needed.

PhoneCam Connect Setup 1.0.0.exe
2

Install Android App

Install PhoneCam from the Play Store. Make sure your phone is on the same WiFi network as your PC.

Search: "PhoneCam Connect" on Play Store
3

Scan QR Code

Open the desktop app — it shows a QR code. Open PhoneCam on your phone → tap Scan QR. Connection is instant.

Or enter IP manually: ws://192.168.x.x:7779
4

Select Camera in Your App

Click Start Webcam in the dashboard. Open Zoom / Teams / Meet → camera settings → select PhoneCam Connect.

Requires native driver for direct app access →

Technical

Built for Reliability

📡 Network & Ports

  • WebSocket (phone frames):7779
  • HTTP discovery page:7780
  • MJPEG stream:7781/stream
  • Named pipe\\.\pipe\phonecam_video
  • TransportLocal WiFi only

🖥️ Desktop App

  • FrameworkElectron 29
  • SecuritycontextBridge
  • IPCWhitelisted channels
  • Windows10 / 11 ✓
  • Mac / LinuxComing Soon

📡 Wire Protocol

  • Video framesJPEG over WebSocket
  • Pipe frame headerPCMF + uint32LE len
  • Quality commandJSON set_quality
  • Backpressure2MB drop threshold
  • Heartbeat10s ping/pong

🎥 Virtual Camera

  • Native driverDirectShow COM
  • CLSID{A1B2C3D4-E5F6...}
  • Linuxv4l2loopback (Soon)
  • MJPEG fallbackAll platforms
  • FFmpeg auto-restart3× retry, 2s delay
