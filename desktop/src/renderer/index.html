Troubleshooting

          

            

              
Phone can't find computer

              
Make sure both devices are on the same WiFi network. Check firewall — allow port 7779 (TCP/UDP).

            

            

              
High latency / laggy video

              
Lower resolution to 720p. Make sure you're on 5GHz WiFi, not 2.4GHz.

            

            

              
PhoneCam not showing in apps

              
Click "Start Virtual Webcam" on the Dashboard. On Linux, install v4l2loopback first.

            

          

        

        

          
Linux Setup (v4l2loopback)

          
Install the kernel module for virtual webcam on Linux:

          

            sudo apt install v4l2loopback-dkms

            sudo modprobe v4l2loopback

            sudo modprobe v4l2loopback devices=1 \

              video_nr=10 \

              card_label="PhoneCam"

              exclusive_caps=1
